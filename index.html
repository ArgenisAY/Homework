¿Qué es un protocolo Http?

El HTTP o HyperText Transfer Protocol es un protocolo de transferencia sobre el que se basa la red informática mundial (WWW). Funciona como base para los intercambios de datos realizados en la web, y mantiene una estructura basadas en los clientes y servidores y orientada a transacciones.

¿Qué es un Query String?

Éste término generalmente se utiliza para hacer referencias a una interacción con una base de datos. Es la parte de la URL que contiene los datos que deben pasar a las aplicaciones web.

¿Qué son los verbos Http?

Son los metodos implementados por el protocolo Http para realizar peticiones al servidor o consumir algun tipo de recurso externo.

¿Qué es un DNS?

El DNS, o sistema de nombres de dominio, traduce los nombres de dominios aptos para lectura humana (por ejemplo, www.amazon.com) a direcciones IP aptas para lectura por parte de máquinas (por ejemplo, 192.0.2.44).

¿Qué es un CDN?

Una red de distribución de contenido (CDN) es un grupo de servidores repartidos en distintas zonas geográficas que aceleran la entrega del contenido web al acercarlo a los usuarios.
